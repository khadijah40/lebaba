## 📖 About

**Lebaba** is a front-end clone of a fashion e-commerce website, built as a hands-on exercise in HTML layout and CSS styling. The project focuses on replicating a modern, multi-section retail storefront with a polished, responsive design — using no frameworks or JavaScript.

The name *Lebaba* reflects a fashion-forward brand identity, and the project demonstrates the ability to structure complex, real-world landing pages from scratch.

## ✨ Features

| Feature | Details |
|---|---|
| 🏠 Hero / Header | Full-width banner with headline and call-to-action |
| 🗂️ Category Grid | 4-category product browsing section with images |
| 🛍️ Product Cards | 7 product listings with images, names, and pricing layout |
| 🔥 Deals Section | Promotional deals banner with visual highlight |
| 📝 Blog Section | 3 blog post cards with images and preview text |
| 📸 Instagram Feed | 6-photo Instagram-style image grid |
| 📱 Responsive Design | Adapts cleanly across desktop and mobile screen sizes |
| 🎨 Pure CSS Styling | No frameworks — all layout and styling hand-written |

---

## 🗂️ Page Sections

The page is divided into the following distinct sections, each styled independently:

1. **Header / Hero** — Brand introduction and main banner image
2. **Category Section** — Browse by category with 4 visual tiles (`category-1` to `category-4`)
3. **Products Section** — Grid of 7 product cards (`product-1` to `product-7`)
4. **Deals Banner** — A promotional section highlighting special offers
5. **Blog Section** — 3 editorial blog cards (`blog-1` to `blog-3`)
6. **Instagram Feed** — 6-image social-style photo gallery (`instagram-1` to `instagram-6`)
7. **Footer** — Navigation links and brand information

## 🚀 Getting Started

No installations or dependencies required. Just clone and open in a browser.

### 1. Clone the repository

```bash
git clone https://github.com/khadijah40/lebaba.git
```

### 2. Open in your browser

```bash
cd lebaba
open index.html
```

Or drag and drop `index.html` directly into any browser window. No server needed.
